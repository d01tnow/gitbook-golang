# go 命令行

## build

```shell
# 基本形式: go build [-o output] [-i] [build flags] [packages]
# 最简单形式. 编译当前目录下的 main 包. 输出的文件名为 mod 名称
go build 
# 显示包名
go build -v
# 显示编译命令
go build -x
# 指定输出程序名称
go build -o app-name 
# 开启竞态检测, 需要 CGO_ENABLED=ON
go build -race
# 并发编译, n = 0, 默认为 CPU 逻辑核数
go build -p n
# 指定文件
go build main.go
# 编译全静态
go build --ldflags '-linkmode external -extldflags "-static”' 
```
